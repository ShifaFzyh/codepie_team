# Dashboard Admin Manajemen Artikel

## Deskripsi
Dashboard Admin Manajemen Artikel adalah aplikasi web yang digunakan untuk mengelola artikel atau blog melalui dashboard admin. Sistem ini memiliki dua role yaitu Admin dan Editor.

## Fitur
- Login
- Role-based Access (Admin & Editor)
- CRUD Artikel
- Dashboard Statistik
- Manajemen User

## Teknologi
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MySQL

## Cara Menjalankan Project
1. Install Node.js
2. Install Express
3. Jalankan server
4. Buka browser di http://localhost:3000
